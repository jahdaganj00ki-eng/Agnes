# Agnes Generation Skill

## Overview

This skill provides guidance for Agnes AI text, image, and video generation workflows.

## Prompt Structure

Use the following structure for image generation prompts:

```
[Subject] + [Scene/Environment] + [Style] + [Lighting] + [Composition] + [Quality Requirements]
```

## Image-to-Image

For image editing, state what should change and what must remain unchanged:

```
[What to change] + [New style/scene] + [Elements to add/remove] + [What to preserve]
```

## Best Practices

- Be specific rather than generic
- Name a camera/lens for photos
- Preserve original composition when editing
- Add negative prompts to exclude unwanted elements
